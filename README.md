# ImageToolbox
This is a Matlab toolbox of Digital Color Imaging Systems.

Reuired knowledges of this toolbox are covered in [ECE638, Purdue University](https://engineering.purdue.edu/~ece638/).

Include: 
## Gamut space
## Fourier Analysis
## Halftone
### Simple threshold dither array
### Cluster dots
### Void and Cluster
### Error diffusion
